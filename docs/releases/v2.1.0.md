# FE Map Creator v2.1.0

FE Map Creator v2.1.0 expands command-line map editing, improves browser file reliability, and adds clearer support and testing workflows.

## Highlights

- Added automation-friendly CLI map creation, editing, resizing, inspection, and `.map`/`.mar`/`.tmx` conversion.
- Added ordered cell, rectangle, flood-fill, lock, terrain, clear-state, and resize operations in versioned job specs.
- Added edit composition with generation, repair, and heterogeneous batch manifests.
- Added stable JSON map inspection plus tileset and terrain discovery.
- Added transactional output handling, TMX image-source rebasing, and stronger overwrite protection.
- Fixed Web app map saving and opening for browser streams that support only asynchronous I/O (#22 and #24).
- Added a structured bug report form and Discussion routing (#26).
- Added downloadable short-lived CI artifacts for Windows, Linux, macOS, and Web builds (#27).
- Added a GitHub Wiki mirrored from the source-controlled project user guide (#28).

## Try it

- [Launch the Web app](https://laqieer.github.io/FEMapCreator/)
- [Read the user guide](https://github.com/laqieer/FEMapCreator/wiki)
- From a desktop archive's `CLI/` directory, run `FE_Map_Creator.Cli.exe map --help` on Windows or `./FE_Map_Creator.Cli map --help` on Linux/macOS.
- Run `tilesets list --json` from the same CLI directory.
