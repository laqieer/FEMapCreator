# FE Map Creator

FE Map Creator is a Fire Emblem-focused map editor and procedural map generator.

- **Web app:** [launch FE Map Creator](https://laqieer.github.io/FEMapCreator/)
- **Desktop downloads:** [latest release](https://github.com/laqieer/FEMapCreator/releases/latest)
- **Source and development:** [GitHub repository](https://github.com/laqieer/FEMapCreator)

## Choose an interface

| Interface | Best for |
| --- | --- |
| Web app | Trying the editor without installing anything |
| Cross-platform desktop GUI | Full visual editing on Windows, Linux, or macOS |
| CLI | Repeatable generation, repair, validation, conversion, and batch workflows |
| [Legacy WinForms editor](Legacy-Editor-Guide.md) | Windows-only tileset processing and specialized legacy workflows |

## Start here

1. Follow [Getting Started](Getting-Started.md) for the Web app, desktop app, or CLI.
2. Use the [GUI User Guide](GUI-User-Guide.md) to create and edit a map.
3. Read [Generation and Repair](Generation-and-Repair.md) before tuning solver settings.
4. Check [Map Formats and Interoperability](Map-Formats-and-Interoperability.md) when exchanging `.map`, `.mar`, or `.tmx` files.
5. See [Troubleshooting and Support](Troubleshooting-and-Support.md) if a workflow fails.
